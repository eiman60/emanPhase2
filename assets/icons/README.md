# Icons Folder

Put your downloaded icon files in this folder.

Use these file names so they are picked up automatically by the app:

- `user.png`
- `wallet.png`
- `notification.png`
- `menu.png`
- `cube.png`
- `rawdah.png`
- `hajj.png`
- `umrah.png`
- `alert.png`

If you want different names, edit `lib/app_icons.dart`.
